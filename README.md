# ⬡ FinScan — SME Financial Health Scanner

🚀 **Live Demo:** [https://finscan-ammksonnmpwjj8nnnf6ixa.streamlit.app/](https://finscan-ammksonnmpwjj8nnnf6ixa.streamlit.app/)

**FinScan** is a real-time risk scoring, anomaly detection, and compliance monitoring dashboard designed for Small and Medium Enterprises (SMEs). Built with **Streamlit** and **Plotly**, it provides a premium, responsive dark-themed interface to scan corporate profiles, evaluate operational metrics, and generate forward-looking projections.

---

## 🌟 Key Features

* **Multi-Dimensional Financial Scoring**: Computes a comprehensive Health Score (0-100) across 5 core categories:
  * **Profitability** (25 points): Based on net margins.
  * **Liquidity** (20 points): Evaluates cash runway and debt ratios.
  * **Compliance** (25 points): Monitors GST status, KYC, audits, and training.
  * **Transaction Health** (15 points): Assesses late payments and transaction volumes.
  * **Operational Maturity** (15 points): Reflects company size, years active, and audit status.
* **Interactive Data Visualizations**:
  * **Score Breakdown**: A custom-styled Radar chart visualizing strengths and areas of risk.
  * **12-Month Projections**: Line and bar forecasts for Revenue, Expenses, and Net Profit.
  * **Historical Trends**: 12-month transaction volumes and flagged item histories.
  * **Interactive Ratios**: Custom gauges representing Debt-to-Revenue and Net Margin percentages.
* **Anomaly & Risk Detection**: Automated indicators that flag high, medium, and low risks (e.g., cash runway stress, large-value transaction spikes, cross-border payment exposure, or late payments).
* **AI-Powered Insights**: Actionable, context-aware suggestions targeting margin pressure, liquidity, transaction health, and compliance postures.
* **Cost Structure Waterfall**: Detailed breakdown of simulated expenses (Payroll, Tech, Marketing, Operations, etc.).

## 🚀 Installation & Local Setup

Ensure you have Python 3.10+ installed. Follow these steps to run the application locally:

### 1. Clone the Repository
```bash
git clone https://github.com/Yashodip2802/FinScan.git
cd FinScan
```

### 2. Set Up a Virtual Environment (Optional but Recommended)
On Windows:
```bash
python -m venv venv
.\venv\Scripts\activate
```
On macOS/Linux:
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Run the Streamlit Dashboard
```bash
streamlit run app.py
```
After executing, the dashboard will open automatically in your browser at `http://localhost:8501`.
